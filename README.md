# 🍎 Apple Clone Website

A modern Apple-inspired landing page built using **HTML**, **CSS**, and **JavaScript**. This project recreates the clean and minimal design style of Apple's website while demonstrating responsive web design and interactive frontend development.

## 🚀 Features

* Clean and modern Apple-inspired UI
* Responsive navigation bar
* Full-screen hero section
* Product showcase cards
* Hover effects and smooth interactions
* Sticky navigation bar
* Mobile-friendly responsive design
* Simple and organized code structure

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla)

## 📂 Project Structure

```text
Apple-Clone/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## ▶️ How to Run

1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Open `index.html` using the Live Server extension or any web browser.
